# Assets

This directory will contain game assets:

## Images (assets/images/)
- ball.png - Ball sprite
- flipper.png - Flipper sprite
- bumper.png - Bumper sprite
- target.png - Target sprite
- table-bg.png - Table background

## Audio (assets/audio/)
- flipper.mp3 - Flipper activation sound
- bumper.mp3 - Bumper hit sound
- target.mp3 - Target hit sound
- launch.mp3 - Ball launch sound
- drain.mp3 - Ball drain sound
- background.mp3 - Background music

Currently using placeholder graphics generated at runtime. Audio uses Web Audio API beeps.

## Asset Sources

Free assets can be found at:
- https://freesound.org (audio)
- https://opengameart.org (images)
- https://itch.io (game assets)
